- Arduino (https://arduino.cc)

- Arduino UNO R3
- Ultrasonic Sensor
- Motor Servo Actuator
- Module WiFi ESP01
- Module Relay Dual Channel
- Mini Pump
- LED
- Jumper Cable
- Breadboard
- Smartphone
